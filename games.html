Prompt - A hardcore platformer made in Java

Description
Prompt is a simple, clean, hardcore platformer. It was made in Java using LWJGL and OpenGL, and is powered by my own custom-built game engine Mote. It makes heavy use of shaders and postprocessing effects to give the impression of an old DOS-era command prompt. The gameplay is not for the faint of heart. Check out the reviews on Steam!

Null Set - In-development RPG and puzzle-exploration game

Description
My current project, a story-based game about punching slimes and the nature of reality. The visual style is meant to reproduce the feel of a game from the PS1, complete with jittery vertices, unfiltered textures, and affine texture mapping. Modern visual effects such as bumpmapping and dynamic shadows are added to create an oppressive, mysterious atmosphere. All art is currently placeholder.

Puddle - Minimalist puzzle game for Android

Description
A test of OpenGL ES on Android. Puddle is a simple swipe-based puzzle game where the goal is to land on every platform in a level. Platforms disappear once you jump off of them, however, and you can't move backwards. It's still in development.
